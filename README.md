# 🎙️ AI Voice Chat

基于 Agent Teams 多智能体协作模式的 AI 语音对话软件。

## 功能

- 🎤 语音输入，实时对话
- 📄 MCP 文档读取与总结
- 🤖 多 Agent 协作（架构师 / 前端 / 后端 / 产品）
- 🚀 CI/CD 自动化部署到 GitHub Pages

## 技术栈

- **前端**: React + Vite + TypeScript
- **后端**: Node.js + WebSocket
- **AI**: GLM-5.1 (智谱 AI)
- **MCP**: 文档读取工具
- **部署**: GitHub Pages + GitHub Actions

## 开发

```bash
cd frontend && npm install && npm run dev
cd backend && npm install && npm run dev
```
